Tested on Google Chrome with Node 8.11.4 installed

To install:
Download the .zip submission.
Unzip to the folder of your choice
Using cmd, navigate to the folder you unzipped and then to "COMP2406F18-A4"
Once cmd is in that directory, enter "npm install" to install the relevant modules
Then enter "node server.js" to start the server.

To run:
As per the server's instructions, go to any of the following URLs to launch the application:
http://localhost:3000/recipes.html
http://localhost:3000/recipes
http://localhost:3000/index.html
http://localhost:3000/
http://localhost:3000

The above will load the default "empty search" results from Food2Fork.

Adding a query string, for example http://localhost:3000/recipes?ingredients=Basil,Cumin
will load the result of searching "basil,cumin" from Food2Fork. 

ISSUES: No known issues.
